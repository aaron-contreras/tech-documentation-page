Serialization

Streams

Data moves in streams from one place to another. Think of a stream as a highway, tunnel or road. There are two types of streams:

  1. Connection streams

    Represent a connection to a source or destination like a

    • File
    • Network sockets

  2. Chain streams

    Allow for a higher level input and output of data but must be chained to a connection stream to connect to a source or destination.

File IO

The Java File IO Class

Lives in the java.io package of course.

A file object represents the pathname (NOT the file contents) of a file or directory and is a MUCH safer way to represent a file (or directory) name than a mere String.

Some really neat methods the File class has:

  • mkdir() -> makes a directory at the pathname the object refers to.
  • list() -> returns a String[] with the names of the files within a directory.
  • getAbsolutePath() -> returns a String with the absolute path of a file or directory.
  • delete() -> deletes a file or directory (returns true if succesful). A directory can only be deleted if it is empty.

Interfaces
